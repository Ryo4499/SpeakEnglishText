# Speak English Text

pyttsx3 sample

## How to run

```sh
pip install pyttsx3
python3 main.py example sentence.
# on wsl
py.exe -m pip install pyttsx3
py.exe main.py example sentence.
# alias
./speak.sh example sentence
# through pass
# rewrite current dir abs path
vim speak.sh 
sudo cp speak.sh /usr/local/bin
```
